# SEO Playbook (90-day)

## Phase 1 (Day 0-30): Stabilize and Measure
- Validate GA4 + GSC + GTM data integrity.
- Repair mismatch between configured IDs and source files.
- Refresh homepage metadata using top-intent GSC terms.
- Confirm event taxonomy for appointment funnel.

## Phase 2 (Day 31-60): Lift Conversion
- Prioritize top-performing service pages from GSC page data.
- Improve snippet relevance for low-CTR but high-impression queries.
- Track appointment event deltas after each metadata refresh.
- Keep technical drift under control (no ID mismatch).

## Phase 3 (Day 61-90): Scale and Defend
- Extend winning patterns to secondary pages.
- Reinforce local search intent and authority signals.
- Lock recurring execution cadence (weekly pull, monthly publish, quarterly strategy refresh).
- Generate next-quarter backlog from measured deltas.

## Required Deliverables per Run
- `analysis.json`
- `plan-90d.json`
- `execution-report.json`
- `rollback-manifest.json`
