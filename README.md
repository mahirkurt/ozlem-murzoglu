# Dr. Özlem Murzoğlu Pediatri Kliniği

Modern pediatri hizmetleri sunan kurumsal web sitesi.

## Teknolojiler

- Next.js 15
- React 18
- TypeScript
- Material Design 3
- Next-intl

## Kurulum

```bash
npm install
npm run dev
```

## Production

```bash
npm run build
npm start
```
